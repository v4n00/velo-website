# Version 1.5.0

`Released 23/OCT/2023`

## Features

- added blindrun simulator
  - you can change a few settings to get different effects
- added rope release indicators
  - you can change their color, thickness and max count
- for local games, you can now also change the player's max speed, grapple hook speed, grapple cooldown, slide cooldown, camera max speed, you can fix RGs and 90s and you can bypass the 300fps framelimit
- you can now set the speedometer to show your last release angle instead
